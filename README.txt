ConfMgr - это командно-строчная утилита для
управления конфигурационным файлом.
Она позволяет добавлять, изменять, удалять и
отображать параметры конфигурации.
Использование
ConfMgr поддерживает следующие команды: add, modify, delete, show.
Каждая команда имеет свои опции.
